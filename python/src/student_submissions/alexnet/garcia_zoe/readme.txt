El siguiente paquete contiene un reporte donde se entrena AlexNet con data de Cifar 10, a través de Keras y TF.

Descomprime el archivo models_Garcia_Zoe.zip para ver el contenido y el reporte.

Los requerimentos (packages y versiones) se encuentran en el texto requeriments.txt

Para instarlos: 
1. Crear un nuevo env
2. pip install -r requeriments.txt

Los modelos pre-entrenados de TF/Keras cuentan con los pesos de Toronto y el modelo en formato h5 listos para ser usados. 
Solo se necesita cambiar el directorio para que estén en la misma carpeta que el archivo ipynb.

El archivo Report  - AlexNet Challenge - Garcia Zoe, contiene datos relevantes a las gráficas obtenidas, observaciones y conclusiones.

Nota: Para poder correr el alexNet_TF_pre es necesario guardar el modelo entrenado en alexNet_TF y descargar los pesos de AlexNet Toronto https://www.cs.toronto.edu/~guerzhoy/tf_alexnet/#:~:text=tune%20the%20networks.)-,bvlc_alexnet.npy,-%2D%2D%20the%20weights%3B%20they
